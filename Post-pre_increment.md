#### Pre-increment
> It increments the value of the variable first and then returns the new value.

```
let x = 5;
let y = ++x; // y is 6, x is 6
```
---
#### Post-increment
> It returns the current value of the variable and then increments the variable.

```
let x = 5;
let y = x++; // y is 5, x is 6
```
